== Beschreibung ==

Mit dem Plugin bietest du in deinen WooCommerce-Shop die Zahlungsart "Kauf auf Rechnung an". Der Vorteil ist, das du mit der "PRO-Version" ebenfalls Bonitätsprüfungen duchführen kannst. Du selbst entscheidest anhand des Schulnotenprinzips, welche Kunden "Kauf auf Rechnung" nutzen dürfen und welche nicht.
 
Wenn diese abgelehnt werden, wird die Kaufoption "Kauf auf Rechnung" deaktiviert und der Kunde kann eine andere Zahlungsart wählen. Schaue dir auch gerne weiter unten das Video an!

= Features =

* Kostenlose Zahlungsart Kauf auf Rechnung für WooCommerce
* Bei bedarf Bonitätsprüfung aktivieren "PRO-Version"
* Bei ablehnung, automatische deaktivierung der Zahlungsart "Kauf auf Rechnung"
* Eigene WooCommerce Staten (Versendet, Rechnung (Mahnung), Rechnung (Export))
* Kunden Erinnerungen für Bestellungen mit Kauf auf Rechnung inkl. automatischer Status änderung
* Keine Vetragslaufzeit oder Grundgebür bei der "PRO-Version"

Bringe ein neues Kauferlebnis für deine Kunden mit ein und profitiere von "Kauf auf Rechnung", der beliebtesten Zahlungsmethode im E-Commerce.

[youtube https://www.youtube.com/watch?v=nc_iBfeZPTI]

Ebenfalls interessant kann das Mahnwesen sein, dafür bieten wir unser Billingportal, welches für dich als Händler komplett kostenlos ist. Die Kombination zwischen dem Plugin „Kauf auf Rechnung“, sowie Billingportal läuft reibungslos und ist ein absoluter Winner! Erfahre mehr über das Billingportal auf der Webseite: https://billingportal.de
